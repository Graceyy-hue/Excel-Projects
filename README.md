# Bank Customer Churn Analysis (Excel)

## Overview
This project presents an exploratory analysis of bank customer churn data using
**Microsoft Excel**. The objective was to identify patterns and factors associated
with customer churn through pivot tables and basic statistical analysis.

## Dataset
- Domain: Banking / Customer churn
- Source: Practice dataset from an online course
- Usage: Educational and learning purposes only

## Tools 
- Microsoft Excel

## Notes
This analysis was completed as part of a learning exercise and focuses on practical
exploration rather than advanced modeling. Feedback is welcome.
